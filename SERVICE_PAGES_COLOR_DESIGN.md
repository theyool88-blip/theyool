# 서비스 페이지 색상 디자인 가이드

**작성일**: 2025-11-23
**목적**: 각 전문 서비스 페이지에 고유한 색상을 부여하여 서비스 특성을 강조하면서도 메인 브랜드(Sage Green)와 조화를 이루는 디자인 구현

---

## 디자인 철학

### 핵심 원칙
1. **차별화된 정체성**: 각 서비스마다 고유한 색상으로 시각적 정체성 부여
2. **브랜드 조화**: 모든 색상이 메인 Sage Green과 자연스럽게 어울려야 함
3. **심리적 적합성**: 서비스 성격에 맞는 색상 심리학 적용
4. **접근성**: 법률 서비스 특성상 전문성과 신뢰감 유지

### 메인 브랜드 컬러
- **Sage Green**: #6DB5A4 (쑥색)
- **사용처**: 홈페이지, 공통 UI 요소
- **특징**: 차분함, 신뢰감, 안정감 (임신 앱 스타일에서 영감)

---

## 서비스별 색상 팔레트

### 1. 위자료 청구 (Alimony)

**색상**: Warm Amber Gold
**Gradient**: `from-amber-600` → `to-yellow-600`
**Hex**: #D97706 → #CA8A04

#### 선정 이유
- **금전적 보상**: 골드/앰버는 가치와 금전을 상징
- **정의 구현**: 따뜻한 황금색은 정당한 권리 회복의 느낌
- **접근성**: 따뜻하면서도 권위있는 색조로 의뢰인에게 친근함과 전문성 동시 전달

#### 색상 조화 분석
- **Sage Green과의 관계**: 유사색(Analogous) - 색상환에서 인접한 위치
- **심리적 효과**: 보상과 성취감을 연상시키면서도 공격적이지 않음
- **가독성**: 흰색 텍스트와 높은 대비로 우수한 접근성

#### 적용 영역
- Hero 섹션 배경 그라데이션
- 프로세스 단계 번호 뱃지
- 카테고리 필터 선택 상태
- CTA 섹션 배경

---

### 2. 재산분할 (Property Division)

**색상**: Cool Cyan-Teal
**Gradient**: `from-cyan-600` → `to-teal-600`
**Hex**: #0891B2 → #0D9488

#### 선정 이유
- **공정성과 균형**: 시원한 청록색은 객관성과 합리성 상징
- **명료함**: 명확한 분배를 의미하는 투명하고 깨끗한 색감
- **신뢰**: 법률적 공정성을 시각적으로 표현

#### 색상 조화 분석
- **Sage Green과의 관계**: 삼원색 조화(Triadic) - 균형잡힌 색상 관계
- **심리적 효과**: 이성적 판단과 공정한 절차를 연상
- **대비**: Sage Green의 따뜻함과 대조되어 객관성 강조

#### 적용 영역
- Hero 섹션 배경 그라데이션
- 프로세스 단계 번호 뱃지
- 카테고리 필터 선택 상태
- CTA 섹션 배경

---

### 3. 양육권 및 양육비 (Custody)

**색상**: Soft Purple-Pink
**Gradient**: `from-purple-500` → `to-pink-500`
**Hex**: #A855F7 → #EC4899

#### 선정 이유
- **부모의 사랑**: 부드러운 퍼플-핑크는 양육과 돌봄의 따뜻함 표현
- **아이의 미래**: 밝고 희망적인 색감으로 긍정적 미래 암시
- **감성적 접근**: 법률적 문제이지만 아이 중심의 따뜻한 관점 강조

#### 색상 조화 분석
- **Sage Green과의 관계**: 보색 조화(Complementary warmth) - 차가운 녹색과 따뜻한 보라-분홍의 균형
- **심리적 효과**: 보살핌, 애정, 안정감을 동시에 전달
- **유연성**: 양육권(퍼플)과 양육비(핑크)의 이중 의미 내포

#### 적용 영역
- Hero 섹션 배경 그라데이션
- 프로세스 단계 번호 뱃지 (6개 단계)
- 카테고리 필터 선택 상태 (4개 카테고리)
- CTA 섹션 배경

---

### 4. 불륜 및 상간 사건 (Adultery)

**색상**: Deep Indigo-Slate
**Gradient**: `from-indigo-700` → `to-slate-700`
**Hex**: #4338CA → #334155

#### 선정 이유
- **진지함과 권위**: 깊은 인디고-슬레이트는 무게감과 법적 엄정함 표현
- **신뢰 회복**: 어두운 톤이지만 냉정하고 전문적인 대응 암시
- **감정적 안정**: 불륜이라는 민감한 주제를 다루면서도 냉철한 법적 접근 강조

#### 색상 조화 분석
- **Sage Green과의 관계**: 유사 한색 계열(Analogous cool) - 쿨톤 통일로 전문성 강화
- **심리적 효과**: 침착함, 권위, 법적 확실성
- **대비**: 가장 어두운 톤으로 다른 서비스와 명확히 구분되며 주제의 무게감 반영

#### 적용 영역
- Hero 섹션 배경 그라데이션
- 프로세스 단계 번호 뱃지 (5개 단계)
- 카테고리 필터 (단일 카테고리 + 추가 FAQ)
- CTA 섹션 배경

## 브랜드 일관성 유지 전략

### 공통 UI 요소는 Sage Green 유지
- **헤더 네비게이션**: Sage Green 호버 효과
- **Footer**: Sage Green 링크 컬러
- **공통 버튼**: Sage-600 기본값
- **플로팅 위젯**: Sage Green 유지

### 서비스 페이지 내에서만 고유 색상 적용
- **Hero 섹션**: 각 서비스 고유 그라데이션
- **프로세스 뱃지**: 해당 서비스 색상
- **FAQ 필터**: 선택 시 서비스 색상
- **CTA 섹션**: 서비스 색상 배경

### 페이지 간 전환 시 자연스러움
- 홈페이지(Sage Green) → 서비스 페이지(고유 색상) 전환이 부드럽게 느껴짐
- 모든 색상이 Sage Green과 조화롭기 때문에 이질감 없음
- 같은 레이아웃 구조로 일관성 유지

## 접근성 (WCAG 2.1 AA 준수)

### 색상 대비 확인
- **모든 그라데이션**: 흰색 텍스트와 최소 4.5:1 이상 대비
- **버튼**: 흰색 배경 버튼으로 명확한 가독성
- **링크**: 기본 텍스트 대비 충분한 색상 차이

### 색맹 고려
- 색상만으로 정보를 전달하지 않음
- 아이콘과 텍스트 함께 사용
- 호버/포커스 상태 명확하게 표시

## 결론

각 서비스 페이지는 이제 다음과 같은 강점을 갖게 되었습니다:

1. **시각적 차별화**: 사용자가 어떤 서비스 페이지에 있는지 즉시 인식 가능
2. **심리적 적합성**: 각 법률 서비스의 특성과 어울리는 색상 사용
3. **브랜드 일관성**: Sage Green 메인 브랜드와 조화로운 색상 체계
4. **전문성 유지**: 법률 서비스 특성에 맞는 신뢰감 있는 색조
5. **사용자 경험**: 페이지 간 전환 시 자연스러운 색상 흐름

이 디자인 시스템은 법무법인 더율의 브랜드 아이덴티티를 강화하면서도, 각 전문 분야의 특성을 효과적으로 전달합니다.
