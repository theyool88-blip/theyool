## 1. Mocha Mousse 기본 정보

### Pantone 17-1230 Mocha Mousse - 2025 올해의 색상

**공식 색상 코드**
- **Pantone**: 17-1230 TCX
- **Hex**: `#A47764` (공식)
- **RGB**: `rgb(164, 119, 100)`
- **CMYK**: `C:0 M:27 Y:39 K:36`

**색상 설명 (Pantone 공식)**
> "Mocha Mousse는 grounding yet luxurious, comforting, chocolatey and warm한 색상입니다. 따뜻하고 깊은 브라운으로, 미묘한 그레이 언더톤이 안정감과 우아함을 동시에 전달합니다."

**브랜드 적합성**
- 법무법인 더율의 로고 색상 `#a8826f`와 매우 유사
- 신뢰성과 전문성을 상징 (금융, 법률, 럭셔리 리테일에 적합)
- 2025년 트렌드를 선도하는 색상으로, 현대적이면서도 시대를 초월한 느낌
- 따뜻함과 접근성을 동시에 전달 (이혼 법률 서비스에 이상적)

## 2. 블로그 전문가 제안 분석 및 완전 채택

### 블로그 권장: 모카무스와 어울리는 6가지 색상

1. **Sand Beige (샌드베이지)** - 차분한 분위기, 따뜻한 배경
2. **Terracotta (테라코타)** - 감각적 대지색, 세련된 CTA
3. **Deep Olive (딥 올리브)** - 고급스러운 자연색, 성공 표현
4. **Sky Blue (스카이 블루)** - 신선한 대비, 신뢰와 투명성
5. **Golden Mustard (골든 머스타드)** - 빈티지 감성, 경력 강조
6. **Charcoal Gray (차콜 그레이)** - 세련된 중후함, 전문성

### 법무법인 더율 적용 전략

**채택 이유**
- **신뢰감** (스카이 블루, 차콜 그레이): 법률 서비스의 필수 요소
- **따뜻한 접근성** (샌드베이지, 테라코타): 어려운 상황의 고객에게 위로
- **성공/안정 표현** (딥 올리브): 승소와 문제 해결의 느낌
- **전문성** (골든 머스타드, 차콜 그레이): 12년 경력, 법조계 권위

**기존 팔레트와의 혁신적 차이**
```
❌ 기존 (평범하고 일반적):
- Copper #FF9A4D → 너무 밝은 주황, 저렴해 보임, 법률 서비스에 부적합
- Trust Blue #6490A4 → 무난하지만 특색 없음, 구식
- Sage Green #5A9976 → 평범한 초록, 차별화 부족
- Cream #FFFCF9 → 너무 밝음, 따뜻함 부족
- Gray #111827 → 너무 차가움

✅ 신규 (블로그 기반, 세련되고 고급):
- Terracotta #D85D42 → 세련되고 감각적인 CTA, 고급스러움 ⭐
- Sky Blue #6EB4D1 → 더 신선하고 현대적, 희망과 투명성 ⭐
- Deep Olive #6B7F59 → 고급스럽고 자연스러운 성공 표현 ⭐
- Sand Beige #FAF9F7 → 따뜻하고 차분한 배경 ⭐
- Charcoal Gray #2A2E35 → 세련된 중후함, 법조계 전문성 ⭐
- Golden Mustard #E5B84A → 빈티지 고급, 경력 배지 ⭐
```

## 3. 법무법인 더율 색상 팔레트 (완전 재설계)

### 3.1 Primary (메인 색상) - Mocha Mousse System

**Pantone 17-1230 기반 10단계 스케일**

```
Mocha 50   #FAF7F5   rgb(250, 247, 245)  // 가장 밝은 배경 (따뜻한 화이트)
Mocha 100  #F4EDE8   rgb(244, 237, 232)  // 섹션 배경 (따뜻한 크림)
Mocha 200  #E9DBCE   rgb(233, 219, 206)  // 카드 배경 (연한 모카)
Mocha 300  #D4C3B5   rgb(212, 195, 181)  // 비활성 요소
Mocha 400  #BDA28D   rgb(189, 162, 141)  // 보조 텍스트
Mocha 500  #A47764   rgb(164, 119, 100)  // 로고 메인 (Pantone Mocha Mousse) ⭐
Mocha 600  #8D6652   rgb(141, 102, 82)   // 버튼 호버
Mocha 700  #7D5E4D   rgb(125, 94, 77)    // 강조 텍스트
Mocha 800  #5F4939   rgb(95, 73, 57)     // 헤더/푸터 텍스트
Mocha 900  #453426   rgb(69, 52, 38)     // 다크 모드 배경
```

**용도**
- Mocha 50-200: 배경, 섹션 구분, 따뜻한 분위기 연출
- Mocha 500: 로고, 브랜드 아이덴티티, 주요 강조, 아이콘
- Mocha 600-700: 보조 버튼, 링크, 인터랙티브 요소
- Mocha 800-900: 헤더, 푸터, 다크 텍스트

Sky 600    #5A9AB5   rgb(90, 154, 181)   // 호버 상태
Sky 700    #487A92   rgb(72, 122, 146)   // 링크

**용도**

**색상 심리학**
- **기존 Trust Blue보다 우수한 점**:
  - 더 밝고 신선함 → 현대적이고 활기찬 이미지
  - 하늘색 연상 → 희망, 투명성, 개방성
  - 법무법인의 새로운 이미지 → 접근 가능하고 친근함
  - 대비가 좋음 → 배경과의 시각적 분리가 명확

**용도**:
- 상담 신청 버튼 (10분 무료 진단 받기)
- 긍정적 액션 유도 (예약 확정, 신청하기)
- 중요한 강조 요소

**기존 Copper와의 혁신적 차이**
| 구분 | Copper #FF9A4D (기존) | Terracotta #D85D42 (신규) |
|------|----------------------|--------------------------|
| 느낌 | 너무 밝고 오렌지색 | 세련되고 대지적 |
| 이미지 | 저렴한 느낌 | 고급스러움 |
| 법률 서비스 적합성 | 낮음 (너무 캐주얼) | 높음 (전문적) |
| 모카무스와의 조화 | 보통 | 완벽 (블로그 추천) |
| 접근성 (대비율) | White 위 2.8:1 (부족) | White 위 4.1:1 (AA) |

**테라코타의 장점**:
- 자연스럽고 따뜻함 (대지색, earth tone)
- 법률 서비스에 적합한 세련됨
- 블로그 전문가 추천 → 2025 트렌드
