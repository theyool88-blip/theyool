### 데스크톱 호버 효과

```
┌──────────────┐
│ [추천]       │
│              │

    ↕️ 0px 위로

```

│ [추천] ●     │  ← 뱃지 점 깜빡임

## 반응형 브레이크포인트

### Mobile (< 768px)
- 그리드: 2 columns
- 간격: 12px (gap-3)
- 카드 높이: min-h-[140px]
- 이모지: 48px (text-5xl)
- 제목: 14px (text-sm)
- 설명: 12px (text-xs)

### Desktop (≥ 768px)
- 그리드: 4 columns
- 간격: 16px (gap-4)
- 카드 높이: min-h-[140px]
- 이모지: 48px (동일)
- 제목: 14px (동일)
- 설명: 12px (동일)

**일관성 유지:**
- 모바일/데스크톱 타이포 동일
- 카드 내부 비율 일관
- 호버/액티브만 차별화
