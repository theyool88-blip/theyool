## 시각적 개선 핵심 6가지

### 1. 모달 크기 및 배경 강화
- **모달 크기**: 4xl → 5xl (더 넓은 작업 공간)
- **배경**: 반투명 블랙 + 블러 효과 (전문성)
- **그림자**: shadow-2xl (깊이감)

### 2. 타이포그래피 계층 구조
- **제목**: xl → 2xl (+20%)
- **섹션 헤더**: lg → xl + 볼드 + 아이콘
- **라벨**: sm → sm 볼드
- **입력 필드**: 기본 → text-base

### 3. 탭 네비게이션 혁신
- **번호 배지**: 원형 배지로 진행 단계 시각화
- **사진 개수**: 실시간 업로드 개수 표시
- **하단 인디케이터**: 활성 탭 하이라이트
- **비활성 상태**: 회색 처리로 명확한 구분

### 4. 섹션 카드화
- **배경 분리**: 각 섹션을 회색 배경 카드로
- **아이콘**: 의미있는 이모지 추가
  - 📋 기본 정보
  - 👤 의뢰인 정보
  - 📝 스토리
  - 📊 추가 정보
  - ⚙️ 게시 설정
- **설명 텍스트**: 각 섹션의 목적 명시

### 5. 체크박스 카드형 디자인
- **배경**: 화이트 카드 + 보더
- **호버 효과**: amber 보더로 선택 가능 표시
- **설명 추가**: 각 옵션의 의미 명확화
- **크기 증가**: w-4 h-4 → w-5 h-5

### 6. 증빙 사진 그리드
- **반응형**: 1열 → 2열 → 3열
- **썸네일**: 128px → 192px (+50%)
- **카드 스타일**: 보더 + 라운드 + 그림자
- **삭제 버튼**: 호버시만 표시 (실수 방지)

## 색상 시스템

### Amber (브랜드 컬러)
- `amber-600`: 주요 버튼, 활성 탭
- `amber-50`: 주의 배경
- `amber-100`: 배지, 하이라이트

### Gray (중립)
- `gray-50`: 섹션 배경
- `gray-200`: 보더
- `gray-700`: 라벨
- `gray-900`: 제목

### 시맨틱
- `red-500`: 필수 표시
- `green-600`: 완료 버튼

## 주요 버튼 개선

### Before
```tsx
<button className="px-4 py-2 bg-amber-600 text-white rounded-lg">
  저장
</button>
```

### After
```tsx
<button className="px-6 py-2.5 bg-amber-600 text-white font-semibold
                   rounded-lg hover:bg-amber-700 shadow-md hover:shadow-lg
                   transition-all flex items-center gap-2">
  <span>→</span>
  저장 후 증빙 추가
</button>
```

**개선점**:
- 패딩 50% 증가
- 그림자 효과
- 아이콘 추가
- hover 애니메이션
