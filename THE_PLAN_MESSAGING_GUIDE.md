## 📌 전략 개요

### 핵심 메시지
**"당신의 불안을 아는 변호사, 답을 가진 법무법인"**

더율은 단순한 법률 서비스가 아닌, 이혼의 전 과정을 미리 설계하는 체계적인 솔루션입니다.

### 설득 논리 흐름

```
공감 (Q&A) → 전문성 (집중) → 증명 (실적) → 차별화 (The Plan)
```

1. **공감**: "당신의 모든 질문에 이미 답이 있습니다" (Q&A 제공)
2. **전문성**: "이혼과 상간, 오직 이것만 합니다" (전문 분야 집중)
3. **증명**: "1,200번의 경험이 만든 노하우입니다" (검증된 전문성)
4. **차별화**: "그래서 우리는 미리 설계합니다" (The Plan)

### 감정적 호소 포인트

- **불안 해소**: "혼자가 아니에요, 우리가 먼저 걸어본 길이에요"
- **신뢰 구축**: "숫자가 말하는 진짜 전문성"
- **희망 제시**: "당신의 새로운 시작, 우리가 설계합니다"
- **확신 전달**: "이미 검증된 승소 전략이 있습니다"

### 작성 원칙

#### 1. 데이터 기반 신뢰성
구체적인 숫자를 전면에 내세워 신뢰도 구축:
- 76개 Q&A
- 21개 성공사례
- 18개 칼럼
- 1,200건 경험
- 12년 전문
- 87% 승소율
- 92시간 집중 투입

#### 2. 공감적 언어
감정적 연결을 만드는 표현:
- "당신의 불안"
- "혼자가 아니에요"
- "우리가 먼저 걸어본 길"
- "막막한 마음"
- "새로운 시작"

#### 3. 전문적 확신
전문성을 강조하는 단어:
- "이미 검증된"
- "체계적인"
- "전략적"
- "미리 설계"
- "승소 전략"

#### 4. 희망적 마무리
긍정적 미래 제시:
- "새로운 시작"
- "함께 걸어요"
- "준비합니다"
- "설계합니다"

### 문체 가이드

**주어**: '우리'와 '당신'을 균형있게 사용
```
❌ "저희가 도와드리겠습니다"
✅ "우리가 함께 걸어갑니다"

❌ "고객님의 사건을 맡겠습니다"
✅ "당신의 새로운 시작을 설계합니다"
```

**문장 구조**: 짧고 명확한 문장 + 감정적 호소를 위한 긴 문장 조화
```
짧은 문장 (임팩트): "이것이 더율의 차이입니다."
긴 문장 (설득): "76개의 이혼큐레이션 Q&A를 통해 당신이 궁금한 모든 것에 미리 답을 준비했습니다."
```

**어조**: 전문적이되 차갑지 않고, 따뜻하되 과하지 않은 중립적 온기
```
❌ 너무 딱딱함: "본 법인은 이혼 전문 법률 서비스를 제공합니다"
❌ 너무 캐주얼: "이혼 걱정 뚝! 우리가 해결해드려요!"
✅ 적절한 톤: "이혼과 상간만 12년째, 그래서 더 잘합니다"
```

## 🌐 더플랜 세부 페이지 연계 전략

### 메인 섹션 → 상세 페이지 흐름

#### 1. 역할 분담
- **메인 페이지 (The Plan 섹션)**: 호기심 유발 + 신뢰 구축 (Why)
- **상세 페이지 (/the-plan)**: 구체적 방법론 + 실행 계획 (How)

#### 2. 일관성 유지
- **비주얼**: 다크 그라데이션 + 프리미엄 느낌 일관성
- **메시지**: "미리 설계하는 승소 전략" 핵심 메시지 반복
- **구조**: 4단계 프로세스를 상세 페이지에서도 확장 설명
- **증거**: 메인의 숫자들을 상세 페이지에서 구체적 사례로 풀어내기

### 상세 페이지 예상 구성

#### 섹션 1: 도입부
```
THE PLAN의 탄생 배경
- 1,200건의 축적된 노하우
- 왜 이혼도 "계획"이 필요한가
```

#### 섹션 2: 4단계 전략
```
Step 1: 진단과 분석 (무료 상담 단계)
Step 2: 증거 설계 (수집 전략 수립)
Step 3: 시나리오 준비 (모든 경우의 수 대비)
Step 4: 실행과 대응 (유연한 전략 조정)
```

#### 섹션 3: 차별화 포인트
```
다른 로펌과의 구체적 비교
- 일반 로펌: 사건 접수 후 진행
- 법무법인 더율: 사건 시작 전 설계
```

#### 섹션 4: 성공 사례
```
THE PLAN으로 승소한 구체적 케이스
- 위자료 5억 확보 사례
- 재산분할 0원 방어 사례
- 양육권 단독 확보 사례
```

#### 섹션 5: CTA
```

```

### 애니메이션
- **지속 시간**: 500ms
- **이징**: `ease-in-out`
- **변경 속성**: `max-height` (0 ↔ 800px) + `opacity` (0 ↔ 1)

### 색상 (재확인)
- 섹션 배경: `bg-gradient-to-b from-blue-50 via-purple-50/30 to-white`
- 카드 배경: `bg-gradient-to-br from-gray-900 to-gray-800`
- 항목 기본: `bg-white/5 border border-white/20`
- 항목 활성: 색상별 (예: `bg-blue-500/10 border-2 border-blue-400`)

### 타이포그래피
```
섹션 제목: text-5xl md:text-7xl font-black
서브 타이틀: text-lg md:text-xl font-semibold

본문: text-sm md:text-base leading-relaxed
링크: text-sm text-[색상]-400 hover:text-[색상]-300
```
