## 이혼 전문 변호사 사이트의 핵심 목표

### 1차 목표: 상담 전환율 극대화
- 방문자를 상담 신청자로 전환
- 전화, 카카오톡, 폼 등 다양한 접점 제공
- 즉각적인 응대 시스템

### 2차 목표: 신뢰 구축
- 전문성 입증 (경력, 성공사례, 수상 내역)
- 투명성 제공 (비용, 절차, 과정)
- 고객 만족도 증명 (후기, 평가)

### 3차 목표: 불안감 해소
- 이혼 절차 안내
- 자주 묻는 질문 (FAQ)
- 예상 비용 안내
- 소요 기간 안내

### 4차 목표: 정보 제공 & SEO
- 법률 정보 칼럼
- 이혼 관련 가이드
- 검색 유입 증대

## ❌ 현재 계획에서 빠진 중요한 것들

### 1. 상담 관련 (전환율 극대화)

#### 🔴 긴급도: 매우 높음

**A. 실시간 상담 채널**
- **카카오톡 채널 연동**: 즉시 상담 버튼 (헤더 고정)
- **네이버 톡톡**: 네이버 검색 유입 시 즉시 대응
- **클릭투콜**: 모바일에서 터치 한 번에 전화 연결
- **채팅 위젯**: 24/7 자동 응답 (기본 FAQ)

**현재 상태**: ❌ 없음
**필요성**: ⭐⭐⭐⭐⭐ (필수)
**이유**:
- 이혼 상담은 긴급성이 높음
- 방문자가 고민 중일 때 즉시 연결해야 전환
- 경쟁사는 대부분 카카오톡 상담 제공

**구현 방법**:
- 카카오톡 채널 개설 후 JavaScript SDK 연동
- 헤더와 하단에 고정 버튼 배치
- 모바일에서는 플로팅 버튼

---

**B. 무료 상담 신청 강조**
- 메인 CTA를 "무료 상담 신청"으로 강조
- "30분 무료 초기 상담" 명확히 표시
- 상담 가능 시간대 표시

**현재 상태**: ⚠️ 상담 신청 폼만 계획
**필요성**: ⭐⭐⭐⭐⭐
**개선 필요**:
- "무료" 강조
- 상담 가능 시간/요일 표시
- 평균 응답 시간 안내

---

**C. 상담 프로세스 안내 페이지**
- 상담 신청 → 접수 확인 → 초기 상담 → 계약 → 진행 → 완료
- 각 단계별 소요 시간
- 준비 서류 안내

**현재 상태**: ❌ 없음
**필요성**: ⭐⭐⭐⭐
**이유**:
- 불안감 해소
- 투명성 제공
- 예상 가능한 프로세스로 신뢰 구축

---

### 2. 정보 제공 (불안감 해소)

#### 🟠 긴급도: 높음

**A. FAQ (자주 묻는 질문)**
- 이혼 절차는 어떻게 되나요?
- 비용은 얼마나 드나요?
- 기간은 얼마나 걸리나요?
- 위자료는 얼마나 받을 수 있나요?
- 양육권은 어떻게 결정되나요?
- 재산분할은 어떻게 하나요?
- 상간소송은 무엇인가요?
- 혼자 오는 게 부담스러운데 가족과 함께 와도 되나요?
- 비밀 보장이 되나요?

**현재 상태**: ❌ 없음
**필요성**: ⭐⭐⭐⭐⭐ (필수)
**이유**:
- 가장 많이 검색되는 키워드
- SEO에 매우 유리
- 사용자의 궁금증 즉시 해소
- 상담 신청 전 의사결정 도움

**구현**:
- `/faq` 페이지
- 아코디언 UI (펼침/접기)
- 카테고리별 분류 (비용, 절차, 위자료, 재산분할, 양육권)
- 관리자 CMS에서 CRUD 가능

---

**B. 이혼 절차 가이드**
- 협의이혼 vs 재판이혼 차이
- 단계별 절차 (소 제기 → 조정 → 심리 → 판결)
- 필요 서류
- 예상 기간
- 각 단계에서 변호사의 역할

**현재 상태**: ❌ 없음
**필요성**: ⭐⭐⭐⭐
**이유**:
- 법률 용어가 어려워하는 일반인 대상
- 절차 이해 → 신뢰 구축 → 상담 신청
- SEO 효과 (절차 검색 많음)

**구현**:
- `/guide/divorce-process` 페이지
- 인포그래픽 형태로 시각화
- 단계별 상세 설명
- 관련 칼럼 링크

---

**C. 서비스 상세 페이지들**
- `/services/alimony` - 위자료 청구 상세
- `/services/property` - 재산분할 상세
- `/services/custody` - 양육권 상세
- `/services/adultery` - 상간사건 상세

각 페이지:
- 해당 서비스 정의
- 관련 법률 근거
- 진행 절차
- 성공 사례
- 예상 비용/기간
- 자주 묻는 질문
- 상담 신청 CTA

**현재 상태**: ⚠️ 홈페이지에 간략하게만
**필요성**: ⭐⭐⭐⭐
**이유**:
- 각 서비스별 깊이 있는 정보 제공
- SEO (키워드별 랜딩 페이지)
- 사용자 니즈에 정확히 대응

---

**D. 수임료 안내 상세 페이지**
- 투명한 비용 구조
- 협의이혼 vs 재판이혼 비용 차이
- 추가 비용 발생 조건
- 결제 방법
- 분할 납부 가능 여부

**현재 상태**: ⚠️ 홈페이지에 간략히만
**필요성**: ⭐⭐⭐⭐
**이유**:
- 비용은 가장 큰 의사결정 요소
- 투명성 = 신뢰
- 타 법무법인과 비교 가능

**구현**:
- `/fees` 페이지
- 테이블 형태로 명확하게
- "문자로 상세 안내 받기" 버튼

## 사용자 여정 (User Journey) 분석

### 시나리오 1: 긴급 상담이 필요한 사용자
```
Google 검색 "이혼 변호사"
→ 홈페이지 방문
→ [카카오톡 상담 버튼] 발견 ← 💡 즉시 추가 필요
→ 즉시 연결
→ 상담 예약
```

### 시나리오 2: 정보를 찾는 사용자
```
Google 검색 "위자료 얼마"
→ FAQ 또는 칼럼 랜딩 ← 💡 FAQ 필수
→ 정보 습득
→ 신뢰 구축
→ "성공사례" 클릭
→ 상담 신청
```

### 시나리오 3: 비교 검토 중인 사용자
```
여러 법무법인 비교
→ "수임료 안내" 확인 ← 💡 상세 페이지 필요
```
